QuickNotes is an Electron-based desktop note-taking application that allows users to create, edit, organize, and manage notes efficiently. The application provides customization options, note organization tools, and document export features to improve productivity and user experience.
 New Features Added
Feature 1: Text Color Customization
Built by: Bohora Bikash (2024591034)
Description:
This feature allows users to change the color of note text using a color picker. The selected color is applied instantly and saved for future sessions.
